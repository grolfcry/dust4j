dust4j

dust.js is the very nice and ultra-fast JavaScript templating library now chosen and adopted by LinkedIn from over 20 others.

dust4j provides extensible, zero-dependency and non-intrusive automatic on-the-fly compilation for dust.js templates for any Java Web Application that adheres Java Servlet specification. If you're not thick from density of buzzwords per character follow me :P

Features

  • small and independent
  • integrates with Spring's DelegatingFilterProxy
  • extensible
  • integrated internal cache and ETag support
  • configurable

Requirements

none, except plain Java SDK 6+, Servlet API and human
